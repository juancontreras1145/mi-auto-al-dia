# Mi Auto al Dia v1.7

Version compacta orientada a utilidad:

- Inicio sin mapa grande ni listas largas.
- Radar usado solo como resumen de zonas.
- Atencion muestra solo los 3 puntos importantes.

- Partes sin categoria "Todos" para evitar saturacion.
- Mapa visual como pantalla/modal aparte.
- Mantiene compatibilidad con respaldos JSON anteriores.
