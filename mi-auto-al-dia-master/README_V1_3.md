# Mi Auto al Día v1.3

Cambios:
- Mapa independiente para categoría Motor usando una imagen original tipo diagrama del vano motor.
- Líquido de frenos agregado y ubicado a la derecha del mapa Motor.
- Los puntos pueden guardar posición distinta para mapa Motor y mapa Vehículo.
- Etiquetas visibles en puntos cuando se filtra por categoría.
- Doble toque/doble clic en mapa ampliado cambia zoom.
- Se mantiene compatibilidad con respaldos JSON de v1.0, v1.1 y v1.2.
- Icono APK actualizado estilo auto 4 puertas.

Versión Android:
- versionName 1.3
- versionCode 4
