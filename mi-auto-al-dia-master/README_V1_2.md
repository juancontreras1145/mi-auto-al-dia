# Mi Auto al Dia v1.2

Cambios principales:

- Tema general negro mas limpio.
- Inicio y pantalla Partes mas compactos.
- Selector de categorias para reducir puntos visibles en el mapa:
  - Todos
  - Motor
  - Ruedas
  - Frenos
  - Luces
  - Documentos
  - Exterior
  - Otros
- Mapa ampliado con etiquetas visibles en cada punto.
- Mapa ampliado con botones Frente / Centro / Atras.
- En modo Solo revisar, la imagen se puede arrastrar para ver zonas fuera de pantalla.
- En modo Editar/agregar, se pueden crear puntos tocando la imagen y moverlos arrastrando.
- Categorias normalizadas para agrupar partes parecidas.
- Nuevo icono de APK estilo auto caricatura de 4 puertas.
- Version Android: versionName 1.2 / versionCode 3.

Compatible con respaldos JSON v1.0 y v1.1.
